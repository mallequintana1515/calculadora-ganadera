# 🥛 Calculadora Ganadera - Sistema de Gestión de Producción de Leche

Una aplicación web en Python usando Flask para calcular la producción de leche, ingresos y ganancias de un ganado lechero.

## 📋 Descripción

Esta calculadora ganadera te permite:
- Calcular la producción total de leche por día
- Estimar ingresos basados en el precio por litro
- Calcular ganancias netas considerando gastos diarios
- Obtener proyecciones mensuales y anuales
- Interfaz web moderna y responsiva

## 🚀 Instalación

### Requisitos
- Python 3.6 o superior
- pip (gestor de paquetes de Python)

### Instalación
1. Clona o descarga este repositorio
2. Navega al directorio del proyecto
3. Instala las dependencias:

```bash
pip install -r requirements.txt
```

## 📖 Uso

### Ejecutar la aplicación web
```bash
python app.py
```

La aplicación se ejecutará en: **http://localhost:5000**

### Interfaz Web
- Abre tu navegador web
- Ve a http://localhost:5000
- Ingresa los datos de tu ganado:
  - Número de vacas
  - Litros de leche producidos por vaca por día
  - Precio por litro de leche
  - Gastos diarios totales
- Haz clic en "Calcular" para ver los resultados

### Características de la interfaz
- ✅ Diseño moderno y responsivo
- ✅ Validación en tiempo real
- ✅ Cálculos automáticos
- ✅ Resultados detallados con proyecciones
- ✅ Interfaz intuitiva con emojis
